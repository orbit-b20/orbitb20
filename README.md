# Orbit B20

**Built natively on Base**

---

## About
Orbit B20 is an open‑source protocol and ecosystem developed exclusively for the Base network. It is designed with security, efficiency, and transparency as core principles.

## Key Features
- Optimized architecture for the Base environment
- Fully open‑source and auditable codebase
- Flexible structure for long‑term development
- Fast transactions with low operational costs

## Technology Stack
- **Network**: Base
- **Language**: Solidity
- **Compatibility**: EVM‑compatible
- **Version Control**: Git & GitHub
